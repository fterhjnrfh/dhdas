================================================================================
                    DH Client App - Windows 7 部署说明
================================================================================

一、系统要求
--------------------------------------------------------------------------------
1. Windows 7 SP1（64位）或更高版本
2. 已安装 KB2533623 和 KB3063858 更新（关键！）
3. Visual C++ 2015-2022 Redistributable (x64)

二、前置条件安装步骤（在 Windows 7 上执行）
--------------------------------------------------------------------------------

步骤 1：确保 Windows 7 已安装 SP1
   - 检查方法：右键"计算机" -> "属性" -> 查看是否显示 "Service Pack 1"
   
步骤 2：安装必要的 Windows 更新
   a) 安装 KB2533623（通用 C 运行时）
      下载地址：https://www.microsoft.com/zh-cn/download/details.aspx?id=26767
   
   b) 安装 KB3063858（UCRT 更新）
      下载地址：https://www.microsoft.com/zh-cn/download/details.aspx?id=49093
   
步骤 3：安装 Visual C++ Redistributable 2015-2022 (x64)
   下载地址：https://aka.ms/vs/17/release/vc_redist.x64.exe
   
   如果上述链接失效，请访问：
   https://learn.microsoft.com/zh-cn/cpp/windows/latest-supported-vc-redist

三、运行程序
--------------------------------------------------------------------------------
直接双击 DH.Client.App.exe 即可运行。

四、常见问题排查
--------------------------------------------------------------------------------

问题 1：DllNotFoundException: Unable to load DLL 'libSkiaSharp'
原因：缺少 Visual C++ Redistributable
解决：安装步骤 3 中的 VC++ 运行时

问题 2：程序无法启动，提示缺少 api-ms-win-crt-*.dll
原因：缺少通用 C 运行时 (UCRT)
解决：安装步骤 2 中的 Windows 更新

问题 3：程序启动后白屏或闪退
原因：可能缺少某些系统更新
解决：
   a) 运行 Windows Update 安装所有重要更新
   b) 确保已安装 .NET Desktop Runtime 6.0（本版本为自包含版本，通常不需要）

问题 4：HTTPS 连接失败
原因：Windows 7 默认不启用 TLS 1.2
解决：程序已自动配置支持 TLS 1.2，如仍有问题：
   a) 安装 KB3140245 更新
   b) 或手动启用 TLS 1.2（注册表设置）

五、文件清单
--------------------------------------------------------------------------------
本发布包包含以下关键文件：
- DH.Client.App.exe          主程序
- libSkiaSharp.dll           SkiaSharp 本地渲染库（必需）
- SkiaSharp.dll              SkiaSharp .NET 封装
- Avalonia.*.dll             Avalonia UI 框架
- 其他依赖库...

六、技术支持
--------------------------------------------------------------------------------
如遇到其他问题，请收集以下信息反馈：
1. 完整错误信息（截图或文字）
2. Windows 版本信息（运行 winver 命令）
3. 已安装的 Visual C++ Redistributable 版本
